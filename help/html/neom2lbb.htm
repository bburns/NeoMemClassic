Folders and Objects

Navigating

To navigate around a NeoMem file, you simply click on the item you want to view in one of the three navigation views on the left side of the screen: Data View, Search View, or Index View. Information about that object will be displayed on the right side of the screen in different views. Different classes of objects may have different views available. For more information on views, see the Views topic.

If an object has a plus sign next to it in the Data View, that means it contains child objects. You can see its child objects by clicking on the plus sign to expand the object. You can also double click on the object to expand it. To "collapse" an object in the Data View, click on the minus sign next to it or double click on it.

You can also navigate around the file using the keyboard - from any view, you can use the arrow keys with the Alt key held down to select a new object on the left side of the screen. For instance, if you are typing something in the Text View, you can hit Alt+Down to go to the next object in the Data View, or whatever view happens to be visible on the left. You can jump to the top of the list with Alt+Home, or jump to the bottom with Alt+End. To expand or collapse items with children, you can hit Alt+Right or Alt+Left.

To switch between views using the keyboard, you can use Alt+PgDn and Alt+PgUp to cycle between the visible views, or type Alt plus the letter that is underlined in the view's name (for instance, Alt+D will take you to the Data View).

Note that to select an object you only need to click on it once.
