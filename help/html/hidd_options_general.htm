Options Dialog

The Options Dialog lets you set various options for the program. Click on the tabs at the top of the dialog to see the different options.

Display Tab:

Default Font The Default Font will be used in the Text View for any new objects that you add. Click on the Change Font button to select a new font.

Name Format This determines how peoples' names are displayed and sorted (first then last or last name first).

Date Format Dates are displayed according to the settings in the Control Panel. The Date Format button brings up the Regional Settings control panel to let you modify the date format.


File Tab:

Default Folder This lets you set the default folder which NeoMem uses for opening and saving files.

AutoStart By checking this option NeoMem will automatically run whenever Windows is started. It will be loaded before any programs in your StartUp folder are run. This is useful if you prefer to always have NeoMem available when you are using your computer.

AutoLoad By checking this option NeoMem will automatically reload the last file opened. Useful if you always work with the same file.

AutoRecover Selecting this option will tell NeoMem to save AutoRecover information in the interval specified. If NeoMem shuts down unexpectedly (due to an error or power outage) the AutoRecover file will be loaded the next time you attempt to load the file.

AutoBackup If this option is checked NeoMem will keep a copy of the current file as a Backup when performing a File Save operation. The backup file will be named with "_Backup" appended to the file name.
