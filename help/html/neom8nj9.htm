Edit Person Name Dialog

The Edit Person Name Dialog lets you modify the value for a person's name. Each person name is composed of a Title, First name, Middle name(s), Last name, Nickname, and Suffix.

You can enter more than one middle name by separating them with spaces and entering them in the Middle name textbox.

You do not have to enter quotes around the person's nickname.

You can change the way NeoMem displays people's names in the Options Dialog.
