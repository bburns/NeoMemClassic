Search View

The Search View allows you to do global searches across all the objects in the current file.

Enter the text you are searching for and hit Enter or click on the Start Search button , and the objects that contain the text specified will be shown in a list. You can scan through the objects by clicking on the Next Match button , or hitting F3. At any point you can edit text in the Text View or change properties in the Contents View or Properties View.

You can switch to the Search View quickly by hitting Alt+S.
