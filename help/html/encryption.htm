Password Protection/Encryption

To password protect and encrypt a file, select File / File Security... from the main menu. This will bring up the File Security Dialog, which will let you select the type of encryption you want and enter a password to protect the file.

Currently NeoMem offers only one type of encryption (RSA RC4).

Use longer passwords for more security, as they are harder to guess by brute force methods.

To turn off password protection and encryption, just select an encryption type of None.
