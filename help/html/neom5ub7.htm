Editing an Existing Class

To edit an existing class, click on the Class Wizard button in the toolbar , or select Object / Class Wizard... from the main menu. This will bring up the class wizard which will guide you through editing an existing class.

You can modify an existing class's Name, Description, Name Property Type, Icon, which Properties are associated with it, and the Default Class for objects of the class.

Note: The changes will only apply to the current file!

For more information, see the Class Wizard topic.
