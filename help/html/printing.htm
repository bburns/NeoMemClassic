Printing Objects

You can print the Text View, the Contents View, or the Properties View of an object. Select the object that you want to print, then select File / Print... or File / Print Preview from the main menu. You can also use the Print or Print Preview buttons on the main toolbar. In the dialog that appears, select the view that you want to print.

You can print just a portion of the Text View by selecting some of the text before printing.

When printing the Contents View, the text will be sized so that all columns will fit on the printed page.

To change the orientation of the printout, select File / Page Setup... from the main menu, and select Portrait or Landscape.

To change the print margins, select File / Page Setup... from the main menu.
