Example File

The example file included with NeoMem (Example.neo) is a good source of information on how to set up and use the program. It includes a tutorial that demonstrates many features of the program.

To open the example file, start NeoMem, and in the Start screen, click on the Open Example File button. If the Start screen is not visible, you can bring it up by selecting Help / Show Start Screen from the main menu.
