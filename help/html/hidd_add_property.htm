Add Property Dialog

The Add Property Dialog lets you add a property to the Contents View or Properties View. The list shows all the properties in the current file, with their Name, Property Type, and Link Source (if the property is a Link property). Select the property that you want to add to the view and hit OK.

Before you add a new property to your file, make sure it doesn't already have the property you want by clicking on the Show all properties option to see all the properties defined in your file.

All actions will only affect the properties in the current file.

Note: If the currently selected object (i.e. the object that is selected on the left side of the screen) is a Folder, then the changes to the Contents View will be saved to that Folder object. If the currently selected object is any other type of object, then the changes will be saved to the class of that object. For instance, if you are inserting a column for a Book object, then all other Book objects in your file will get the new column also.
