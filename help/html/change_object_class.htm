Changing the Class of an Object

Each object in a NeoMem file is associated with a class. This class determines which icon is displayed with the object, what properties are displayed in the Properties View, and what views are available for the object. You can change the class of an object by right clicking on it and selecting Change Class... from the popup menu (or clicking on the object and selecting Object / Change Class... from the main menu). This will bring up a dialog showing all the classes available in the current file - you can select the new class and hit OK, and the object will be changed to the new class.

Note: If the new class has different properties associated with it than the old class, and you had already entered information for the old class's properties, you can still access them through the Properties View. If they are no longer applicable for the object, you can delete them by clicking on the property value and hitting the Delete key.
