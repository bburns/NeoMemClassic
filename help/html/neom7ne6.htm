Folders and Objects

Changing the Icon of an Object

Each object has an icon associated with it that is determined from its class. For instance, all Book objects might have an icon of a book. However, you can override the default icon for an individual object by right clicking on the object and selecting Change Icon... from the popup menu (or clicking on an object and selecting Object / Change Icon... from the main menu). This will bring up a dialog which will show all of the icons available to choose from. You can also create a new icon by clicking on the Import... button, which will let you select a Windows icon file to import (a file with a .ico extension).

Note: Any icon that you import will be stored in the current file only, and will not be available to other files unless you import it to them as well.

Changing Icons for a Class of Objects

To change the icon used to represent a class (in other words, so all objects of a class get a different icon), you can right click on an object and select Change Class Icon... from the popup menu (or click on the object and select Object / Change Class Icon... from the main menu). This will let you change the icon used to represent all objects of the same class. If any of the objects of the class have overridden their icons, then their icons will not change. You can also accomplish the same thing with the Class Wizard. See the Edit Class topic for more information.
