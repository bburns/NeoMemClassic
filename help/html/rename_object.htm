Renaming an Object

You can change the name of an object in several different ways:

  • Click on the object and hit F2 to edit the name in place

  • Click on the object and hit F4 to bring up a dialog

  • Click on the object and click on the Rename toolbar button

  • Right click on the object and select Rename

  • Click on the object and select Edit / Edit Value in Place from the main menu

You can also change an object's name in the Contents View by typing in a new value and hitting Enter, Tab, or an arrow key. To edit the existing name just hit F2.

Note: Names do not need to be unique, even if you are using a link property and link to an object with a duplicate name.
