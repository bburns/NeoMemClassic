File Security Dialog

The File Security Dialog lets you modify the security settings for the current file. It can be accessed from the File / File Security... menu option or from the File / Save As... dialog's Options... button.

Select a type of encryption and enter a password to password-protect and encrypt the file.

Encryption The type of encryption to use when encrypting this file. Currently NeoMem offers only one type of encryption.

Password Enter a password, up to 16 characters, to protect the file. Longer passwords provide more protection from brute-force cracking methods. Be sure to write this down in a safe place!

Note: To turn off password protection and encryption, just select an encryption type of None.
