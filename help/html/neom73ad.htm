Edit Date Dialog

The Edit Date Dialog lets you modify the value of a date property. You can select a date from the calendar, and enter a time if desired. Use the left and right arrow buttons to change the displayed month. To change the year quickly, you can click on the year and scroll up and down through the year values.

Note: If you are editing a date that has modifiers attached to it (for instance, "5/4/1967?", "March 1953", "1975", "After 1983", etc.), this dialog will remove the modifiers and give you a normal date when you hit OK.

Note: If you edit a time only value (e.g. "5:45:00 PM"), the calendar will show December 31, 1899, which is how a blank date is stored.

The dialog is limited to dates after September 1753.

For more information on the date property type, see Properties and Property Types.
