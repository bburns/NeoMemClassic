Folders and Objects

Moving an Object

You can move an object to a new location in the file in several different ways:

  • Click on the object and select Object / Move Object To... from the main menu

  • Click on the object and click on the Move Object button on the toolbar

  • Click on the object and press Ctrl+M

  • Right click on the object and select Move Object To... from the popup menu

  • Drag and drop the object to the new location (move the mouse pointer over the object you want to move, press the left mouse button down, then drag it to the new location and let the button go)

Note that to move an object from the Contents View, you must drag it from the cell containing the object's icon.

If the location you want to drag the object to is not visible, you can hold the object you are dragging near the top or bottom of the view, and it will scroll up or down for you.

Note: The location of objects does not affect their ability to be linked to by link properties. You can move a folder containing link objects or a linked object to a new location without affecting any current links. For more information, see the Properties and Property Types topic.
