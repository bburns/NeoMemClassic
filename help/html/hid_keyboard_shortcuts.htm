Appendix

Keyboard Shortcuts

The following keyboard shortcuts are available in NeoMem:

Ctrl+G Goto another object
Alt+Down Navigate to next object
Alt+Up Navigate to previous object
Alt+Home Navigate to first object
Alt+End Navigate to last object
Alt+Right Expand current object
Alt+Left Collapse current object
Alt+, Navigate to previous object in history
Alt+. Navigate to next object in history
 
Alt+PageDown Switch to next visible view
Alt+PageUp Switch to previous visible view
 
Objects
Ctrl+W Add a new object
Ctrl+E Add a new folder object
F2 Edit value in place (rename)
F4 Edit value in a dialog
Delete Delete selected object
Alt+Enter Show properties for selected object
Ctrl+M Move selected object to another location
 
Text View
Ctrl+M Move selected text to another object
Ctrl+X Cut text
Ctrl+C Copy text
Ctrl+V Paste (formatted text)
Ctrl+Alt+V Paste Plain (unformatted text)
 
Ctrl+Alt+K Select and apply new background color
Ctrl+Alt+T Select and apply new text color
Ctrl+K Apply current background color
Ctrl+T Apply current text color
 
Ctrl+B Toggle bold formatting
Ctrl+I Toggle italic formatting
Ctrl+U Toggle underline
Ctrl+Space Remove formatting from selection
 
Ctrl+F Find text
F3 Repeat find
Ctrl+H Replace text
Ctrl+Z or Alt+Backspace Undo the last action
F5 Insert current date/time
 
Data View
Numeric Keypad * Expand all items under current item
Numeric Keypad - Close all items under current item
Numeric Keypad + Open current item
 
Views
Alt+D Switch to Data view
Alt+S Switch to Search view
Alt+I Switch to Index view
Alt+T Switch to Text view
Alt+C Switch to Contents view
Alt+P Switch to Properties view
Ctrl+Alt+M Minimize to System Tray

Ctrl+A Select all text
Ctrl+Backspace Delete one word to the left
Ctrl+Delete Delete one word to the right
Shift+Right Arrow Select one character to the right
Shift+Left Arrow Select one character to the left
Ctrl+Shift+Right Arrow Select to the end of a word
Ctrl+Shift+Left Arrow Select to the beginning of a word
Shift+End Select to the end of a line
Shift+Home Select to the beginning of a line
Shift+Down Arrow Select one line down
Shift+Up Arrow Select one line up
Ctrl+Shift+Down Arrow Select to the end of a paragraph
Ctrl+Shift+Up Arrow Select to the beginning of a paragraph
Shift+Page Down Select one screen down
Shift+Page Up Select one screen up
Alt+Ctrl+Page Down Select to the end of a window
Ctrl+Shift+Home Select to the beginning of a document
 
Documents
Ctrl+N Create a new document
Ctrl+O Open an existing document
Ctrl+S Save current document
