Appendix

System Requirements

Minimum Requirements:

  • Windows 98/Me/NT/2000/XP

  • Intel Pentium or comparable processor

  • 16MB RAM

  • 2MB free hard disk space

  • 800x600 display recommended

  • 16 bit color display recommended (for color icons)

The more memory your system has, the better NeoMem will perform and the more objects and data you will be able to store. See the Limitations topic for more information.
