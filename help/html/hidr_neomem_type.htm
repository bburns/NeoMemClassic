Overview

What is NeoMem?

NeoMem is a program you can use to store all kinds of information - things to do lists, book collections, website passwords, music collections, shopping lists, addresses, and many other kinds of information. It can be thought of as a combination Word Processor and Database, with an interface that is similar to Windows Explorer.

Each object can store text information with a simple word processor view, and you can create different types of objects and create properties that describe them. Each object can also contain any number of child objects. Using the Class Wizard, you can easily create your own classes (object types) and the properties that describe them.

Where to start

If you are just getting started, you might want to take a look at the example file included with the program. It includes several examples of the types of information you can store with the program and a tutorial on various features.

You can also read the overviews on various topics, including Folders and Objects, Classes and Properties, and Views.

How does it work?

You can think of each object in the file as being a box into which you can put any number of other objects and property values. Child objects are displayed in the Contents View, and the property values are displayed in the Properties View.
