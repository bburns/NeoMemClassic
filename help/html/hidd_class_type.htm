Class Wizard: Name Type

In order to let you store people as objects, NeoMem gives you the option of specifying the Name Type for a class as a Person Name. This means that when you enter a name for the object, NeoMem will attempt to parse it into first and last names. This lets you sort people objects by last name then first name instead of first then last. You can change the way NeoMem displays people names in the Options dialog.

If the class you are creating or editing is not a person class, just use Regular Name option.
