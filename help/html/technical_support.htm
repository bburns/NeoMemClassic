Technical Support

Questions

If you run into any questions using NeoMem, please check through this help file - the FAQ in this help file and on the website lists some frequently asked questions and answers. If you cannot find an answer to your question, please send me an email at bburns@neomem.org. The FAQ on the website will also be updated as any new questions arise.

Bugs

If you run into any bugs or other problems, it's possible that a newer version of NeoMem has fixed them - you can find information on the latest version at the website: www.neomem.org.

If the problem has not been addressed, please send me an email with as much information as possible in order to help track down the problem - your system type, Windows version, what was happening when the problem occurred, etc.
