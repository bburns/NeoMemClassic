Properties and Property Types

Every object in a NeoMem file can store any number of property values. Each NeoMem file contains a collection of properties which can be assigned to the objects in the file. Each property can be of one of the different property types, summarized below.

Each property type may display a different type of dialog when you hit F4 or double click on a property value. For example, if you hit F4 on a Link property value, the Edit Link Dialog is displayed.

Type Description Edit Dialog
   
Boolean A simple Yes or No value. (none)
Currency Stores a currency value (e.g. $1,531.59). Values are stored with 15 digits of precision. Windows Calculator
Date/Time Can store a date and an optional time value, in the range of January 1, 100 to December 31, 9999.

You can store a date and a time by entering a string like "May 5, 1973 5:45pm", or a time alone, like "8:30:00 PM".

You can also store approximate dates, for example "May 1973", "1956", "Aft June 10, 1953", "Bef 1650", "Ca. 1254", "1979?", "Summer 1954", "Christmas 1982", etc., and NeoMem will attempt to store the date so that it will sort correctly.

If the date you enter cannot be parsed and stored numerically, you will be asked if you want to store the date as a string. For example, you could enter "Christmas early 70's". Such a string will not sort correctly, however.

 Edit Date Dialog

Note: This dialog is limited to dates after September 1753.
Email Use to store email addresses Edit String Dialog
File Use to store links to local files File Selection Dialog
Folder Use to store links to local folders Folder Selection Dialog
Hyperlink (Deprecated in favor of Email and Website property types). Use to store email addresses and websites to allow for quick launching of email program or browser. Hyperlink values will be shown in green, and the cursor will change to a hand when the mouse is over a value. Clicking on the value will launch your email program or web browser, as appropriate.

Note: A "mailto:" or "http://" prefix is not necessary.

 Edit String Dialog
Link Links to one or more objects in the same file. For instance, a book object could link to one or more author objects using a link property. Edit Link Dialog
Number Stores a number value (e.g. 5,233.54). Numbers are stored in double-precision format, with 15 digits of precision and a range of approximately 1.7E-308 to 1.7E+308. Windows Calculator
String Stores a string value (e.g. "This is a string"). Strings are basically unlimited in length (up to 2GB). Edit String Dialog
Website Use to store website addresses Edit String Dialog

Note: If you need to change the property type of an existing property, you can do so by editing the property using the Property Wizard. If you have already stored values for the existing property, NeoMem will attempt to convert the values to the new property type, though this conversion may be of different quality for different types of conversions. NeoMem will warn you before you attempt such a conversion.
