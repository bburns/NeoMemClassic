Edit Property Dialog

The Edit Property Dialog lets you edit the definition of a property, which defines the type of data it will store and how it will be displayed.

Name The name of the property. This does not have to be unique, but it will help to distinguish different properties if it is.

Description Optional description of the property.

Type The type of data that the property will store. Possible types are Boolean (Yes/No), Currency, Date/Time, Link, Number, and String. See Properties and Property Types for more information.

Type Description This displays the description for the selected property type. This field is read-only.


If you select the type Link, the following additional options will become available:

Link Source A link property links to other objects in the file. The Link Source is the folder (or other object) which contains the objects you want to link to. For instance, an Author property could specify an Authors folder for the Link Source, which would contain Person objects representing authors.

To create a new Link Source folder, click on the New button next to the dropdown.
Limit Link to One Object Link properties can link to more than one object. If you want to limit links for this property to one object, check off this option.

Display Link Object Hierarchy If your link source folder contains a hierarchy of items you can display the parents of the link object by checking this option. For instance instead of just displaying "Taos" for the property value, NeoMem could display "Taos, New Mexico, Texas, United States". The example file contains a Locations folder which demonstrates this option.

Display Additional Property Sometimes the names of the objects that you link to might not be unique. In this case you can display an additional property to make the name unique. For instance, if you were linking to a folder containing museums, and some of the museums had the same name, you could distinguish them by displaying the Location as well. In this case, "Museum of Fine Arts" would become "Museum of Fine Arts, Boston".

Property This dropdown becomes available if you check off the Display Additional Property above. Select the property you want to be displayed after the name of the link object.
