Folders and Objects

Changing the Contents of an Object

Each object can specify the default class to use when adding new subobjects to it. For instance, in the example file, the Books folder has the Book class as its default class, so that when you add a new object to the Book folder, it will know to add a Book object.

You can change the default class for any folder or object by right clicking on it and selecting Change Contents... from the popup menu (or clicking on the object and selecting Object / Change Contents... from the main menu), then selecting the class that you want to use as the default.

Note: You are not limited to adding objects of the default class - you can always select a different class for the new object.

Changing Contents for a Class of Objects

You can also change the default class associated with an object's class by right clicking on an object of the class and selecting Change Class Contents... from the popup menu (or clicking on the object and selecting Object / Change Class Contents... from the main menu). For instance, this would let you modify the default class for all Book objects to a Chapter class, except for those Books which you had already modified their default class using Change Contents. In other words, an object's default class will override the object class's default class.

For more information, see the topics Adding a New Object, and Adding a New Class.
