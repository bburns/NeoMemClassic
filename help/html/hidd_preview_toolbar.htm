Print Preview

The Print Preview screen shows how the printout will look. You can walk through the pages by clicking the Prev and Next buttons. Clicking on the Two Pages button will let you see two pages at once.

The margins are displayed as dotted gray lines - to change these, close the print preview screen and select File / Page Setup from the main menu, then enter new values for the margins. By default the margins are set to 0.75 inches - if you change them NeoMem will remember the new settings.

For the Contents View, the font size is adjusted so that all the columns will fit horizontally. If the font size is too small, you can adjust it by resizing columns so they are smaller, removing columns, or by changing to Landscape mode (close the print preview screen, select File / Page Setup from the main menu, choose Landscape orientation, and redo the print preview).

To send the output to the printer, click on the Print... button - if you only want to print certain pages, you can enter a page number range in the dialog that appears.

Note: The text size in the preview may be slightly off from the actual printout.
