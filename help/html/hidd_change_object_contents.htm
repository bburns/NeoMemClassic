Change Contents Dialog

The Change Contents Dialog lets you change the default class of an object, which is the class used as the default when adding a new child object.

For instance, if you set the default class of a folder to the Book class, the next time you add an object to the folder using the New Object Dialog it will default to adding a new Book object. Also, when you use the New Object Row in the Contents View (marked by a triangle), the class of the new object will be the Default Class for its parent (in this case, a Book).
