Folders and Objects

Adding a New Object or Folder

You can add new objects or folders by many different methods. A folder is just another type of object, but there are special commands for adding folders because they are slightly different from regular objects.

Adding Objects

To add a new object, you can do one of the following:

  • Select Object / New Object... from the main menu

  • Click on the New Object toolbar button:

  • Type Ctrl+W

  • Right click on an object and select New Object... from the popup menu

You will see the New Object dialog, which will ask you for the name and class of the new object. You can also specify the location for the new object in the Location dropdown, which defaults to the currently selected object, or the object you right clicked on. Click on OK or hit Enter and the new object will be added to your file. See the New Object Dialog for more information.

Adding Folders

To add a new folder, you can do one of the following:

  • Select Object / New Folder... from the main menu

  • Click on the New Folder toolbar button:

  • Type Ctrl+E

  • Right click on an object and select New Folder... from the popup menu

  • Add a new object and select the class "Folder".

In the New Folder dialog you can specify the class of objects the folder will contain by default, by selecting the class from the Contents list. See the New Folder Dialog for more information.

Contents View

Another way to add a new object is through the Contents View. The last row of the Contents View has a triangle next to it - this is the new object row. You can click on a cell in this row and start typing a value. When you hit Enter, Tab, or an arrow key, a new object will be created. See the Contents View for more information.

Changing Class

If you add an object and realize that it is not the right class, you can easily change the class of the object by right clicking on it and selecting Change Class.... See the Change Class topic for more information.
