Change Class Icon Dialog

The Change Class Icon Dialog lets you change the icon for a specified class, so that all objects of the class will use the specified icon. However, if you have set the icon for any individual object, that will take precedence over the setting for the class. Select the icon you want to use for all objects of the class and hit OK.

Import... To add a new icon to the current file, you can import a standard Windows Icon file (a file with a ".ico" extension) by clicking this button. You will be asked for a name for your icon, and your new icon will then show up with all the standard NeoMem icons.

Edit... This will let you rename the currently selected icon.

Delete... This will let you delete the currently selected icon. Note that the predefined NeoMem icons cannot be deleted.
