Class Wizard: Properties

This step lets you specify which properties will be associated with the class you are creating or editing. The properties that you check off in this screen will show up in the Properties View for any objects of this class.

Each object gets some properties predefined that cannot be unchecked - Name, Class, and Description. Each property in the file can be used by any number of classes.

You do not have to specify all the properties for your class in this step - you can always come back later and add more, or add new properties to an object by adding columns to the Contents View.
