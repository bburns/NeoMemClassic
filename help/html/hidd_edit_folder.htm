New Folder Dialog

The New Folder Dialog lets you add a new folder object to the file. Enter a name, an optional description, select the default contents, and hit OK to create your new folder object.

Width1Width3Width1548Width3Width5760Name A name should be a short description of the folder - the name is what shows up in the Data View, on the left side of the screen, so shorter names usually work best.

Note: Names do not have to be unique - you can have two objects with the same name, even if one of them is being linked to by a link property.
