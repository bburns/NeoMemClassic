Property Wizard

The Property Wizard lets you manage the properties stored in the current file. You can create new properties and edit or delete existing ones. Each property can be used by any number of classes.

Before you add a new property to your file, make sure it doesn't already have the property you want by clicking on the Show all properties option to see all the properties defined in your file.

Note: You can also add or edit properties using the Class Wizard, but the Property Wizard is sometimes helpful when you need to make a quick change to a property.

Show all properties Select this option to show all properties available in the current file.
