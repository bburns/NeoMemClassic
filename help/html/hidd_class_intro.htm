Class Wizard: Intro

The Class Wizard lets you add, edit, or delete classes for the current file. Each object in a file is of a certain class. The class determines what icon an object will have, what properties are associated with the object, how the name will be parsed, if the children will be sorted automatically, etc.

To start the Class Wizard, click on the Class Wizard button on the main toolbar (bmc btnClassWizard.bmp), or select Object / Class Wizard... from the main menu.

Note: All changes made to classes only apply to the current file!

Add To add a new class, select the option Add a new class, then click on the Next button to go to the next step of the wizard.

Edit To edit a class, select the option Edit or delete an existing class, select the class you want to edit from the list, then click on the Next button.

Delete To delete a class, select the option Edit or delete an existing class, select the class you want to delete from the list, then click on the Delete button. When you're done, click on the Cancel button to exit the class wizard.


Note: For all your input to be saved properly, you must complete all the steps of the wizard, and click on the Finish button in the last step.
