Folders and Objects

Home Folder

Each new NeoMem file starts with a Home Folder. This folder cannot be deleted, but you can rename it if you wish. This folder can contain any number of other folders or objects.

The Home Folder has a special view, the Home View, which displays the file's description and a Tip of the Day.

To change the description for a file, you can right click on the Home Folder and select Object Properties..., then enter a description in the textbox marked Description. This description will then be visible in the Home View whenever you open the file.

You can hide or show the Contents View for the Home Folder by selecting View / Hide Contents View... or View / Show Contents View....
