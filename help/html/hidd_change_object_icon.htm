Change Icon Dialog

The Change Icon Dialog lets you change the icon of an object, which is normally determined from the class of the object. In some cases, you may want to override the icon and use a different icon for an object. Select the icon you want to use for the object and hit OK.

Import... To add a new icon to the current file, you can import a standard Windows Icon file (a file with a ".ico" extension) by clicking this button. You will be asked for a name for your icon, and your new icon will then show up with all the standard NeoMem icons.

Note: To change the icon for all objects of a class, you can right click on an object of the class and select Change Class Icon... from the popup menu, or use the Class Wizard to modify the icon associated with the class. See the Class Wizard topic for more information.
