Change Class Contents Dialog

The Change Class Contents Dialog lets you change the default class for a class of objects, which determines the type of object will be added when adding a new child object.

For instance, if you set the class contents for the Book class to the Chapter class, the next time you add an object to any Book object it will default to adding a new Chapter object. However, if you have specified the contents of any individual object, that will take precedence over the setting of the object's class. For instance, you could have one specific Book object that you have set the contents to be Paper objects, which would take precedence over the setting for the entire Book class.
