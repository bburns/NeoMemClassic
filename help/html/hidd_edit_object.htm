New Object Dialog

The New Object Dialog lets you add a new object to the file. Enter a name, select a class, and hit OK to create your new object. You can add a folder using this dialog and selecting "Folder" for the class or by using the New Folder Dialog.

Name A name should be a short description of the object - the name is what shows up in the Data View, on the left side of the screen, so shorter names usually work best.

Note: Names do not have to be unique - you can have two objects with the same name, even if one of them is being linked to by a link property.
Class Select the class for the new object. The class you select determines which icon will be associated with the new object, what properties will show up in the Properties View, and what views will be available to the object.

You can also create a new class by clicking on the Create new class... button to the right of the list.

Class Description Displays the description of the selected class. This is read-only.

Location Select the location where you want to add the new object.
