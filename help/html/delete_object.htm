Folders and Objects

Deleting an Object

You can delete an object in several different ways:

  • Click on the object and hit the Delete key

  • Click on the object and click on the Delete Object toolbar button

  • Right click on the object and select Delete Object... from the popup menu

  • Click on the object and select Object / Delete Object... from the main menu

NeoMem will check to see if there are any links to the object you are trying to delete. If there are, you will be asked if you want to remove the links before you will be able to delete the object.

If the object you are deleting contains any child objects, all the child objects will be deleted also.

Warning: Once you delete an object, there is no way to restore it, so use this carefully!

The Home Folder cannot be deleted.
