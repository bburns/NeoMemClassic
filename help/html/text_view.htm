Text View

The Text View is a simple word processor, allowing you to store text information with each object.

Features:

  • The Text View can store hyperlinks to websites, emails, files, etc., for instance: www.somewhere.org, mailto:someone@somewhere.org, file:c:\bootlog.txt. The Text View recognizes the following prefixes: http:, file:, mailto:, ftp:, https:, gopher:, nntp:, prospero:, telnet:, news:, wais:.

  • Multilevel Undo and Redo are available for modifications made to the Text View. Use the Edit / Undo and Edit / Redo commands.

  • Both foreground and background colors can be set. Use the toolbar buttons or keyboard shortcuts (see table below) or the menu selections Edit / Set Background Color... and Edit / Set Text Color...

  • The Text View will remember the cursor position for each object so the next time you return to an object the insertion point will be at the same place you left it.

If you want to paste formatted text as plain text, use the Paste Plain command (Ctrl+Alt+V).

To apply formatting to a word it is not necessary to select the entire word first. Just position the cursor in the word and apply the formatting (eg Ctrl+B for bold).

Commands:

Ctrl+M Move selected text to another object
   
Ctrl+Alt+K Select and apply new background color
Ctrl+Alt+T Select and apply new text color
  Ctrl+K Apply current background color
  Ctrl+T Apply current text color
   
  Ctrl+Spacebar Remove formatting from current selection
   
Ctrl+B Bold
Ctrl+I Italic
Ctrl+U Underline
   
Ctrl+X Cut
Ctrl+C Copy
Ctrl+V Paste (formatted)
Ctrl+Alt+V Paste Plain (unformatted)
   
  Ctrl+A Select All
   
  Move text using mouse Select text, click and hold mouse on selection, then drag to new location and release.

Instead of moving the text you can copy it by holding the Ctrl key down during the operation.
   
  Move text to different object using mouse Select text, click and hold mouse on selection, then drag over new object and release. Note: The text will be moved to the end of the new object.

Instead of moving the text you can copy it by holding the Ctrl key down during the operation.
